In v0.3.0

  • Monaco Editor — same engine as VS Code
  • 5 built-in themes (Monaco colors change)
  • File tree with right-click context menu
  • Integrated terminal — pip, node, python, go
  • One-click run for 18+ languages
  • Debug panel UI
  • Optional tools installer — n8n, Flowise, Claude CLI
  • The fahhhh sound — plays on every error
