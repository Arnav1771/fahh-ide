# FAAH Audio Assets

## Audio Files

### faah.mp3
The iconic "FAAH!" sound effect. This should be an MP3 file.

**Specifications:**
- Format: MP3
- Sample Rate: 44.1 kHz or higher
- Channels: Mono or Stereo
- Duration: 0.5-2 seconds
- Size: < 500 KB recommended

**How to add your sound:**
1. Place your MP3 file here and name it `faah.mp3`
2. Rebuild the extension
3. Test with "FAAH: Test Sound" command

**Recommended sources:**
- Create custom sound using Audacity (free)
- Find royalty-free sounds on Freesound.org or Zapsplat.com
- Use your own recorded audio

## Icon File

### icon.png
The extension icon displayed in the VS Code marketplace and extensions panel.

**Specifications:**
- Format: PNG
- Size: 128x128 pixels
- Background: Transparent preferred
- File size: < 50 KB

## Creating Your Own Assets

### Audio File Creation
1. Use Audacity (free): audacityteam.org
2. Record "FAAH!" or similar sound
3. Export as MP3 with 128 kbps bitrate
4. Save as `faah.mp3`

### Icon Creation
1. Use Figma (free tier): figma.com
2. Create 128x128 square canvas
3. Design icon
4. Export as PNG
5. Save as `icon.png`

## File Sizes

Keep files small for faster downloads:
- Audio: Compress MP3 with lower bitrate if needed
- Icon: Optimize PNG using tools like TinyPNG
