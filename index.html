Every LSP diagnostic and build failure triggers fahhhh.mp3. A 3-second atomic cooldown prevents flooding. Configurable in ~/.fahh/config.json. Impossible to ignore.

// what's inside

Everything a real IDE needs.
One thing no IDE has.

Editor
Monaco — the VS Code engine
Full completions, multi-cursor, minimap, and syntax highlighting for 15+ languages. Ctrl+S writes to disk via Rust. 5 built-in themes.
LSP
Language servers, auto-detected
rust-analyzer, tsserver, pylsp, gopls, clangd, jdtls — auto-found on PATH. Full go-to-definition, hover, and error diagnostics in Monaco.
Run
One-click code execution
▶ Run the current file in one click. Python, Node.js, Go, Rust, Java, C++, and more — streamed stdout/stderr in the Run panel. No config needed.
Debug
Step-through DAP debugger
Breakpoints, step over, step in, continue, variables, call stack. Python via debugpy. Node.js/TypeScript via --inspect. Go via Delve.
Sound
fahhhh.mp3 — the core feature
Plays on LSP errors and build failures. 3-second atomic cooldown. Web Audio API. The filename has 4 h's. Do not rename it. This is non-negotiable.
Themes & plugins
5 built-in themes, formatters
Fahh Dark (Catppuccin), Fahh Light, GitHub Dark, Dracula, Solarized. Prettier, Black, gofmt, rustfmt, clang-format integrated as formatters.
